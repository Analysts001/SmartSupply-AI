# 📦 SmartSupply-AI: Predictive Supply Chain Intelligence System

An end-to-end **data-driven supply chain analytics and prediction system** built using SQL, Machine Learning, and interactive dashboards.  
The project helps in analyzing shipment delays, optimizing logistics, and predicting risk in supply chain operations.

---

## 🚀 Project Overview

SmartSupply-AI is designed to:
- Analyze supply chain shipment data using **SQL (BigQuery)**
- Perform **Exploratory Data Analysis (EDA)** in Python
- Build a **machine learning model** to predict shipment delay risk
- Visualize insights using **Power BI / Tableau dashboard**
- Deploy an interactive web app using **Streamlit**

---

## 🏗️ Tech Stack

- **Programming:** Python
- **Database:** Google BigQuery (SQL)
- **Machine Learning:** Scikit-learn (Random Forest)
- **Visualization:** Power BI / Tableau
- **Web App:** Streamlit
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
